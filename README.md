Resumo:
    O trabalho simula uma estação metereológica de forma simples se utilizando do raspberry pi 3.

Componentes:
    -Raspberry Pi 3
    -Sensor de temperatura e umidade: DHC11
    -Barometro BMP085
    -Protoboard
    -LED RGB


Serviços de IOT

    -Ubidots
